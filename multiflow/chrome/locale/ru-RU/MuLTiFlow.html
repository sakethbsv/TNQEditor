Добро пожаловать в MultiFlow

Браузер / Редактор для S & T документов, которые можно открыть / отредактировать слово или HTML5 локальные файлы или просматривать веб-страницы.

x=−b±b2−4ac2a3

Если вы обнаружите, что выше Math не делает правильно (особенно, если корень радикального не распространяется на нижнюю часть застроенной-фракция), пожалуйста, скачайте и установите Stix шрифты из http://www.mozilla. орг / проекты / MathML / шрифты / Стикс / STIXBeta.zip в папку шрифтов операционной системы (в случае Microsoft Windows оно либо Windows / Fonts папки или WINNT / Fonts папки) и перезапустить MultiFlow того, чтобы сделать математику лучше.

Вы можете вводить и редактировать HTML + MathML + SVG, в среде WYSIWYG. Для загрузки файлов использовать HTML5 Файл-> Открыть кнопку выше. Пункт стиль / Стиль символа / MathML в файл HTML5 отображается справа от кнопки ParaStyle выше. Разметки HTML5 из ​​нажатии пункта можно увидеть в окне серые код. Нажатии пункта приобретет белый фон означает, что он доступен для редактирования. CTRL +, CTRL-используется в зум-зум и из выделенного текста. Вставьте кнопка используется для вставки современный TeX-подобные UTN28 вход для генерации HTML + MathML формат (есть также отдельная UTN28 ниже поле для создания MathML). Разметка в серо-окне, расположенном ниже могут быть обновлены до пункта, нажав "Изменить TUXML" кнопку.

Чтобы Обзор, введите ваш адрес в поле справа от голограммы URL и нажмите ALT + b. Если вы нажмете на текст и нажмите Ctrl + Q, то он будет считать вас обратно в редактор из браузера. От редактора HTML5, если вы выделить фрагмент текста и нажмите Ctrl + G, то он будет искать по DuckDuckGo.com для этого слова. Если нажать Ctrl + D, то поиск www.thefreedictionary.com за это. Аналогично Ctrl + R для Google Scholar и Ctrl + W для Википедии. Вы также можете правой кнопкой мыши и выбрать действие, вместо использования горячих клавиш. При нажатии на текст и нажав Ctrl + Q (или ESC?) Приведет вас обратно в документ, который вы редактировали. Новый "Windows" меню для переключения Редактор / Браузер / Scratch. Блокнота на данный момент используется для дампа данных, таких как цитаты из Google Scholar. Для переключения на другие сценарии, например, индийских, Arbaic т.д. можно использовать Google транслитерации виджета доступны в контекстное меню. Для вставки специальных символов использования Unicode в меню правой кнопки мыши всплывающее окно.

SVG разметка поддерживается и векторные элементы могут быть вставлены и редактировать с помощью меню Вставка-> Vector. SVG разметки могут быть отредактированы в сером поле и "Смена TUXML" кнопка обновления изменений. ChemML поддержка находится на пути ...

Ввода / вывода форматов DOCX, ODF / ODT, EPUB которые поддерживаются. Для преобразования LaTeX / XeLaTeX документ XHTML + MathML документ, пожалуйста, используйте утилиту Perl, LaTeXML, которое можно загрузить и установить с http://dlmf.nist.gov/LaTeXML/. Для чтения текста PDF, пожалуйста, используйте команду Linux pdftohtml построен с использованием Poppler библиотеки.

Пакет Debian включает в себя необходимые шрифты Стикс и вы можете скачать последние версии файлов multiflow.deb от http://sourceforge.net/projects/multiflow/
