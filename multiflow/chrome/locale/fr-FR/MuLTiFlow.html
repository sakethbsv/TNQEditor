Bienvenue à Multiflow

Une navigateur/éditeur pour S&T documents qui peut ouvrir/éditer des fichiers Word ou HTML5 local ou naviguer sur le web.

x=−b±b2−4ac2a3

Si vous trouvez que les mathématiques ci-dessus ne rendent pas correctement (surtout si la racine radicale ne s'étend pas jusqu'au fond de la fraction bâtie), alors s'il vous plaît téléchargez et installez les polices STIX de http://www.mozilla.org/projects/mathml/fonts/stix/STIXBeta.zip au dossier de polices de votre système d'exploitation (en cas de Microsoft Windows, il est sous le dossier WINDOWS/FONTS ou le dossier WINNT/FONTS) et redémarrez Multiflow afin de mieux rendre les mathématiques.

Vous pouvez taper et modifier du code HTML+MathML+SVG, dans un environnement WYSIWYG. Pour charger un fichier HTML5 utilisez le bouton Fichier->Ouvrir en dessus. Style paragraphe/style caractère/MathML dans un fichier HTML 5 est affiché à droite du bouton Parastyle en dessus. Le balisage HTML5 du paragraphe cliqué peut être vu dans la fenêtre de code grise en dessous. Le paragraphe cliqué va acquérir un fond blanc indiquant qu'il est modifiable. CTRL+, CTRL- est utilisé pour zoomer et dézoomer le texte en surbrillance. Le bouton Insérer permet d'insérer une entree moderne TeX-like UTN28 pour générer format HTML+ MathML (il y a aussi une boîte séparée UTN28 en dessous pour créer MathML). Le balisage dans la boîte grise en dessous peut être réactualisé dans le paragraphe en appuyant sur le bouton "Changer TUXML".

Pour naviguer sur le web, tapez votre URL dans la boîte à la droite de l'étiquette URL et appuyez sur ALT+b. Si vous cliquez sur le texte et appuyez sur CTRL+q, alors il vous ramènera à l'éditeur du navigateur. De l'éditeur de HTML5, si vous sélectionnez un morceau de texte et appuyez sur CTRL+g, puis il va rechercher pour ce mot en utilisant DuckDuckGo.com. Si vous appuyez sur CTRL+d puis il en recherche www.thefreedictionary.com. De la même, CTRL+r pour Google Scholar et CTRL+w pour Wikipedia. Vous pouvez aussi cliquer à droite et choisisser l'action au lieu d'utiliser les touches de raccourci. Cliquer sur le texte et appuyer sur CTRL+q (ou ESC?) vous ramènera vers le document que vous éditiez. Vous utilisez le menu «Windows» pour commuter entre l'éditeur/navigateur/bloc-notes. Le bloc-notes pour l'instant est utilisé pour déverser des données comme des citations de Google Scholar. Pour passer à d'autres scripts tels que Indic, Arabic etc. on peut utiliser le choix Google Translittération disponible dans le menu clic-droit. Pour insérer des caractères spéciaux utiliser le choix Unicode dans le menu clic-droit.

SVG balisage est soutenu et éléments vectoriels peuvent être insérés et édité en utilisant le menu Insérer->Vecteur. Le balisage SVG peut être édité dans la zone grise et le bouton "Change TUXML" réactualisera les modifications faites. Le soutien de ChemML est en route...

Entrée/sortie des formats DocX, ODF/ODT, ePub sont nativement soutenues. Pour convertir un document LaTeX/XeLateX à un document XHTML+MathML, s'il vous plaît utilisez un utilitaire de perl, LaTeXML, qui peut être téléchargé et installé de http://dlmf.nist.gov/LaTeXML/. Pour lire le texte PDF s'il vous plaît utilisez la commande Linux pdftohtml construite en utilisant la bibliothèque Poppler.

Le paquet Debian inclut les polices STIX nécessaires et vous pouvez télécharger les derniers fichiers multiflow.deb de http://sourceforge.net/projects/multiflow/.
