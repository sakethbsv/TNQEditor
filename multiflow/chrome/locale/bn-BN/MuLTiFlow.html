Welcome to MuLTiFlow

A Browser/Editor for S&T documents that can open/edit Word or HTML5 local files or browse the web.

x=−b±b2−4ac2a3

If you find that the above Math doesn't render properly (especially if the root radical doesn't extend to the bottom of the built-up-fraction) then please download and install the STIX fonts from http://www.mozilla.org/projects/mathml/fonts/stix/STIXBeta.zip to your operating system's font folder (in case of Microsoft Windows it is either WINDOWS/FONTS folder or WINNT/FONTS folder) and restart MuLTiFlow in order to render math better.

You can type and edit HTML+MathML+SVG, in a WYSIWYG environment. To load a HTML5 file use File->Open button above. Paragraph style/Character style/MathML in a HTML5 file is displayed to the right of ParaStyle button above. The HTML5 markup of the clicked paragraph can be seen in the grey code window below. The clicked paragraph will acquire a white background indicating that it is editable. CTRL+, CTRL- is used zoom-in and zoom-out highlighted text. Insert button is used to insert a modern TeX-like UTN28 input to generate HTML+MathML format (there is also a separate UTN28 box below for creating MathML). The markup in the grey-box below can be updated to the paragraph by pressing "Change TUXML" button.

To Browse, type your URL in the box to the right of the URL label and press ALT+b. If you click on the text and press CTRL+q then it will take you back to the editor from the browser. From the HTML5 editor if you select a piece of text and press CTRL+g then it will search using DuckDuckGo.com for that word. If you press CTRL+d then it lookup the www.thefreedictionary.com for it. Similarly CTRL+r for Google Scholar and CTRL+w is for Wikipedia. You can also Right-Click and choose the action instead of using the short-cut keys. Clicking on the text and pressing CTRL+q (or ESC?) will take you back to the document you were editing. A new "Windows" menu for switching Editor/Browser/Scratch. The Scratch pad for now is being used to dump data like citations from Google Scholar. For switching to other scripts such as Indic, Arbaic etc. one can use Google Transliteration widget available in the Right-Click menus. To insert special characters use the Unicode menu in the Right-Click popup.

SVG markup is supported and vector elements can be inserted and edited using the menu Insert->Vector. The SVG markup can be edited in the grey box and the "Change TUXML" button will update the changes made. ChemML support is on the way...

Input/output formats DocX, ODF/ODT, ePub are natively supported. To convert LaTeX/XeLaTeX document to XHTML+MathML document, please use a perl utility, LaTeXML, which can be downloaded and installed from http://dlmf.nist.gov/LaTeXML/. To read Text-PDF please use Linux command pdftohtml built using Poppler library.

The debian package includes the required STIX fonts and you can download the latest multiflow.deb files from http://sourceforge.net/projects/multiflow/
